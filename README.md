## How to Preview Markdown in VS Codium
- Open the instructions-v1.3.md file (.md stands for "markdown", as opposed to "HTML markup" i.e it is simpler)
- Hit Command-Shift-P (macOS) or Control-Shift-P (Windows/Linux) to open the VS Codium control palette
- Type "Markdown Preview" 
- Use the arrow keys (if necessary) to select "Markdown Preview to the Side"
- The Markdown-to-HTML rendered preview will appear in a new tab
